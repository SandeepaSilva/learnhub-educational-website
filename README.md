# LearnHub – Educational Platform (Frontend)

LearnHub is a responsive educational platform interface designed to showcase modern web design, user interface development, and interactive frontend features.

## 🚀 Features

* Responsive multi-section website (Hero, Categories, Courses, Testimonials, Newsletter)
* Sticky navigation bar with scroll effect
* Mobile-friendly hamburger menu
* Smooth scrolling and section highlighting
* Scroll-based animations using Intersection Observer
* Clean and modern UI design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* DOM Manipulation
